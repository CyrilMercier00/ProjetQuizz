##### Lancer les requêtes et tests en série

Cliquez sur la collection, cliquez sur run en haut à droite.
Vous pourrez choisir d'enlever certaines requêtes à éxecuter, de les faire plusieurs ou même rajouter un délai en microsecondes.
Vous pouvez également indiquer un fichier où mettre les données de réponses (utile pour les get par exemple).
